Panda HeartMe Overlay

HOW TO USE WITH GITHUB PAGES

1. Extract this ZIP.
2. Upload the extracted files to your GitHub repo:
   - index.html
   - style.css
   - script.js
   - README.txt

Do NOT upload this ZIP by itself.

3. In GitHub:
   Settings > Pages > Deploy from branch > main > /(root)

4. Your URL should look like:
   https://YOURNAME.github.io/panda-overlay/

5. In TikTok LIVE Studio:
   Add Source > Link
   Paste your GitHub Pages URL
   Resize it over your scene.

TESTING

- The overlay shows demo green panda hearts every few seconds.
- Clicking/tapping the overlay also triggers a heart burst.
- TikFinity connection is attempted automatically, but TikTok LIVE Studio may block local WebSocket connections from a GitHub Pages HTTPS page.
- If live TikFinity events do not work in TikTok LIVE Studio, use OBS Browser Source + OBS Virtual Camera for the most reliable setup.
